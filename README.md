# CalculatorApp
### 實現功能
> 加、減、乘、除  
> 階乘   
> 多項式運算

### Demo
* 加、減、乘、除，答案顯示包含小數與負數，小數點顯示至第六位，第六位後四捨五入

![](https://github.com/BillyHuang1027/CalculatorApp/blob/main/plus_minus_multiple_divide.gif)     

* 階乘功能包含 `0! = 1`

![](https://github.com/BillyHuang1027/CalculatorApp/blob/main/factorial.gif)

* 多項式運算（先乘除後加減）
